An HTML and CSS cheatsheet from FireInsidE

Table tags

TagDescription
table The wrapper element for all HTML tables.
thead Wrapper for the rows that will be the table's headers.
tbody Wrapper for the rows that will be the table's body.
tfoot Wrapper for the footer's rows.
tr Contains a table row.
td Contains data in a table row.

Table attributes

AttributeDescription
colspan Defines how many columns a td element should span.
rowspan Defines how many rows a td element should span.
align Allows to align a table just like using a text-align attribute in CSS.

Some CSS attributes

AttributeDescription
border Creates a border using three parameters (1px solid rgb(x, x, x)) around an item.
